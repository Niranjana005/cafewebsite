**CAFE DELIGHT- Static Website:**
A very simple ,clean and modern HTML & CSS based website wit essential sections for a cafe fully structured for learning web development basics.

**Features:**

**Home Page:** 
Cafe name,tagline and navigation menu

**Menu page:**
Display of dishes of starters,main course,desserts  with simple styling with the usage flex and a smooth scrolling

**Contact page:**
Basic contact information placeholder

**Custom CSS Styling:**
organized layout and color scheme

**Responsive Design:**
Header and navigation bar adjust to screen sizes

**Technogies Used:**

**#HTML5**-structure and content

**#CSS3**-Styling and layout
